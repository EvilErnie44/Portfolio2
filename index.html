Web and Mobile Development

During this course of this module I was tasked with the production of a Mobile First Responsive website, I chose to develop a site for a prospective Web Development studio, the main technologies which I implemented during the course of this production were, HTML 5, CSS3, Javascript and jQuery.

Prior to the development of the Site I had to produce user personas and scenarios as a means to investigate features and services the site would offer to its target audience. I was then required to develop the implemented features and design of the site until I had ensured the outlined pre requisites were met.

Once the site was complete, I took part in a team exercise to develop a testing strategy. This was an enjoyable exercise where all members contributed to the final testing strategy. We were each individually assigned with choosing specific scenarios to create test plans for. I was assigned to develop a test plan for navigation and how media queries affected elements.

Web Project Managment Methods

I was part of a small team which were required to develop an online vehicle sales website, the production took place in using the agile methodology, we held weekly sprints during which the team would need to produce artifacts which we required as detailed in our weekly product backlog.

The developments main pre requisite was that the website had to offer some form of interactive feature. Using personas and user scenarios I help assert what features would need to be present and cascaded this to our team via divergent design sessions.

Over the course of this group collaboration we implemented the use of project managment tools to track and visually represent our progress, this would include the use of gantt charts and trello boards in order to keep track of what was happening and when, finally we used gitHub as a version control suite to maintain our code.
